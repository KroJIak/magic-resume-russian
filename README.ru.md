Magic Resume — это современный онлайн-редактор резюме, который делает создание профессионального резюме простым и приятным. Проект построен на TanStack Start и Framer Motion, поддерживает предпросмотр в реальном времени и настраиваемые темы.

## 🚀 Быстрый старт

1. Клонируйте проект

```bash
git clone git@github.com:JOYCEQL/magic-resume.git
cd magic-resume
```

2. Установите зависимости

```bash
pnpm install
```

3. Запустите dev-сервер

```bash
pnpm dev
```

4. Откройте в браузере `http://localhost:3000`

## 📦 Сборка и деплой

```bash
pnpm build
```

## 🐳 Деплой через Docker

### Docker Compose

1. Убедитесь, что Docker и Docker Compose уже установлены

2. Выполните в корне проекта:

```bash
docker compose up -d
```

Это:

- автоматически соберет образ приложения
- запустит контейнер в фоне
